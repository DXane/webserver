1.)

Lade die bot .jar von der Release-Seite auf Github runter und starte sie den Bot durch den Befehl java -jar packagename.jar.
Vor dem Start fügt in bot.config den API-Token von Discord ein. Ansonsten beendet sich der Bot sofort.

2.)

Beim ersten Starten des Bots erstellt er einen Ordner namens wdatabase. Ihn diesem Ordner wird die h2-Datenbank angelegt auf der Alle Charaktere eingespeichert werden.
In der Repository befindet sich eine Sampledatenbank die man Nutzen kann. Charaktere konnen nach der Bot zuweisung über den Befehl $add hinzugefügt werden. Der Befehl wurde noch nicht implementiert.
